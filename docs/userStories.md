

Users:
Working with radiation at a clinic or other radiation profession
Radiology nurse
Engineer at a power plant
Doctor 
Supervising role who has access to data for all workers for a specific company

User stories:
As a staff member, I want to be able to see my own history of dosimeter readings
One reading per month shown on a graph
Filtered by year
Filtered by Time period 
As a staff member, I want to be able to see what dosimeters I have
As a supervisor, I want to be able to add different dosimeter data
I want to see data for all departments
I want to see data for each department
I want to choose data for a specific role (eg a technician)
I should be able to add departments
I should be able to add personnel
I should be able to assign as many dosimeters as necessary to each staff member
I should be able to set a limit for a specific role
I should be able to set a limit for a specific department
I should get a notification if a limit is exceeded
I want to be able to add data from a nicely formatted source, eg csv
I want to be able to do stats on my data
As a user, I want to be able to export a report of past readings in a nice format


MVP:

Supervisor can add departments
Supervisor can set limits per department
Supervisor can add roles
Supervisor can set limits per role
Supervisor can see data per department, role and worker
